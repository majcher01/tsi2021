LISTY

List używa się do grupowania tekstu. Dane w nich umieszczone są wypunktowane, mogą być też uporządkowane (ponumerowane) lub nie.

Listy uporządkowane

Lista uporządkowana to taka, w której pozycje posiadają swój indywidualny znak (numer bądź litera) i sa zorganizowane według nich.

Listy uporządkowane tworzymy za pomocą znacznika <ol></ol>

Aby stworzyć podpunkty należy zastosować znacznik <li></li>

<ol>
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
</ol>

  1. jeden
  2. dwa
  3. trzy
Element <ol> wskazuje jedynie na rodzaj listy. Wszystkie elementy listy muszą się znaleźć pomiędzy znacznikami <li> oraz </li>.

Możemy zmienić wygląd numeracji poprzez dodanie atrybutu type wraz z odpowiednią wartością.

Gdy jako wartość ustawimy "A" podpunkty listy będą numerowane wielkimi literami alfabetu (A, B, C, D, E, F...)
Gdy jako wartość ustawimy "a" podpunkty listy będą numerowane małymi literami alfabetu (a, b, c, d, e, f...)
Gdy jako wartość ustawimy "I" podpunkty listy będą numerowane cyframi rzymskimi (I, II, III, IV, V, VI...)
Gdy jako wartość ustawimy "i" podpunkty listy będą numerowane małymi cyframi rzymskimi (i, ii, iii, iv, v, vi...)

Możemy rozpocząć numerowanie od dowolnej litery lub liczby. By to zrobić musimy dodać atrybut start, a jako wartość ustawić numer pierwszego podpunku. (gdy np. ustawimy start="5" lista rozpocznie się od 5)

<ol type="A" start="5">
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
</ol>

  1. jeden
  2. dwa
  3. trzy

type="i"

<ol type="i">
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
<li>cztery</li>
</ol>

  1. jeden
  2. dwa
  3. trzy
  4. cztery
Listy nieuporządkowane

Lista uporządkowana to taka, w której pozycje nie posiadają swojego indywidualnego znaku, są wymienione luźno np. od myślników czy kropek.

Listy uporządkowane tworzymy za pomocą znacznika <ul></ul>

Aby stworzyć podpunkty należy zastosować znacznik <li></li>

<ul>
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
</ul>

  • jeden
  • dwa
  • trzy
Tak jak w listach numerowanych mamy możliwość zmiany stylu podpunktów.

Dostępne są opcje:

disc - ○
circle - ●
square - ■
none - nie wyświetli się nic

<ul type="square">
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
</ul>

  • jeden
  • dwa
  • trzy
Tytuły list

Aby poinformować użytkowników czego dotyczy lista warto nadać jej tytuł.
Aby to zrobić można użyć znacznika nagłówka (od <h1> do <h6>)

<h5>Przykładowa lista</h5>
<ul>
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
</ul>

Przykładowa lista
  • jeden
  • dwa
  • trzy

Owoce

</li>
<li>
Warzywa

  2. Warzywa

Co się stanie gdy skończą się litery ?

Gdy lista uporządkowana typu "a" lub "A" będzie miała więcej niż 26 pozycji (ilość liter w angielskim alfabecie) kolejny numer będzie poprzedzała liera "a" lub "A" w zależności od typu listy, np. aa, ab, ac itd. / AA, AB, AC itd. Dwa elementy <li> w liście nie mogą mieć identycznego identyfikatora.

<ol style="font-size: 22px; text-align: left;"
type="A" start="25">
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
<li>cztery</li>
<li>pięć</li>
</ol>

  1. jeden
  2. dwa
  3. trzy
  4. cztery
  5. pięć

Własne ikonki jako punktory do list

HTML pozwala na określenie własnych punktorów w listach, poniżej przykłady.

  • Emotikony

    Aby ustawić emotikon jako punktor należy dodać parametr stylu css.
    list-style-type: X;
    Gdzie X jest kodem wybranego emotikona.

<ul style="font-size: 22px; text-align: left;
list-style-type: '\1F44D';" >
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
<li>cztery</li>
<li>pięć</li>
</ul>

  • jeden
  • dwa
  • trzy
  • cztery
  • pięć
  • Zdjęcie

    Aby ustawić zdjęcie jako punktor należy dodać parametr stylu css.
    list-style-image: url('X');
    Gdzie X jest ścieżką do wybranego zdjęcia.

<ul style="font-size: 22px; text-align: left;
list-style-image: url('../img/majcherski-ico-white.png');" >
<li>jeden</li>
<li>dwa</li>
<li>trzy</li>
<li>cztery</li>
<li>pięć</li>
</ul>

  • jeden
  • dwa
  • trzy
  • cztery
  • pięć
