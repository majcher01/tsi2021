Kursory

Kursor

selektor { cursor: rodzaj; }
Rodzaj odpowiada za kształt, jaki przybierze wskaźnik myszki, po naprowadzeniu go na dany element.
Kursory ogólnego przeznaczenia


  • auto - automatyczny wybór kształtu kursora (zależy od typu elementu, na który wskazujemy - tryb domyślny)
    Najedź myszką na tekst
  • default - kursor domyślny
    Najedź myszką na tekst
  • none - brak kursora
    Najedź myszką na tekst

  • help - kursor pomocy
    Najedź myszką na tekst
  • pointer - wskaźnik (wskazuje odsyłacz)
    Najedź myszką na tekst
  • progress - kursor postępu
    Najedź myszką na tekst
  • wait - "poczekaj"
