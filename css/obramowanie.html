Obramowanie

Obramowanie można określić na dwa sposoby:
  • Określić wszystkie parametry osobno:
    selektor
    {
    border-X-style: styl;
    border-X-width: szerkość;
    border-X-color: kolor;
    }     Gdzie jako "X" należy wpisać które obramowanie określamy (top, bottom, right, left).
  • Lub wszystkie razem:
    selektor
    {
    border-X: styl szerokość kolor;
    }     Gdzie jako "X" należy wpisać które obramowanie określamy (top, bottom, right, left).
Obramowanie górne

selektor { border-top: styl wielkość kolor; }
TUTAJ znajdziesz co powinno się znajdować w miejscach styl, wielkość i kolor.

Styl, wielkość i kolor

Styl obramowania

solid - jednolite ciągłe
dotted - kropkowane
dashed - kreskowane
double - podwójne
groove - wklęsłe
ridge - wypukłe
inset - wklejające
outset - uwypuklające
none - brak obramowania
Rozmiar obramowania
Można tu podać konkretną wartość np. 2px lub jedną ze stałych:

thin - cienkie obramowanie
medium - średnie obramowanie
thick - grube obramowanie
Kolor obramowania
Jako kolor należy podać konkretną wartość w zapisie rgb, heksadecymalnym lub definicję koloru.

red
#FF0000
rgb(255, 0, 0)
