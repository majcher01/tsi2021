Czcionki

Styl czcionki

selektor { font-style: styl; }
Selektorem może być dowolny znacznik, w którym można wpisywać tekst, np. p - akapit, h1 - tytuł czy td - komórka tabeli

Style mogą być następujące:
  • normal - czcionka normalna (podstawowa)
  • italic - czcionka pochylona
  • oblique - również czcionka pochylona
Rodzaj czcionki

selektor { font-family: rodzaj; }
Jako "rodzaj" należy wpisać nazwę czcionki, na przykład:
  • Arial
  • Courier New
  • Georgia
  • Impact
  • Verdana
Wielkość czcionki

selektor { font-size: rozmiar; }
Rozmiar czcionki możemy podać w trzech jednostkach: pt, px lub em
Waga czcionki

selektor { font-weight: waga; }
Jako "waga" należy wpisać nazwę czcionki, na przykład:
  • normal - czcionka normalna (podstawowa)
  • bold - czcionka pogrubiona
  • 100, 200, 300... - waga czcionki
Przykłady wag:
  • 300
  • 400
  • 500
  • 600
  • 700
