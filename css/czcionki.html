Styl czcionki

selektor { font-style: styl; }
Selektorem może być dowolny znacznik, w którym można wpisywać tekst, np. p - akapit, h1 - tytuł czy td - komórka tabeli Jako "styl" należy wpisać:
  • 2014 r. - HTML 5.0
  • 1997/98 r. - HTML 4.0
  • 1997 r. - HTML 3.2
  • 1995 r. - HTML 3.0
  • 1993 r. - Zarys opublikowany przez IETF
