Tekst

Kolor tekstu

selektor { color: kolor; }
Jako kolor należy podać definicję koloru

Przykład:

HTML:
<p>Tekst 1</p>

CSS:
p {color: red;}

Rezultat:
Tekst 1
Dekoracja tekstu

selektor { text-decoration: dekoracja; }
Jako "dekoracja" należy wpisać:
  • none - bez dekoracji
  • underline - podkreślenie
  • line-through - przekreślenie
  • overline - nadkreślenie
  • blink - migotanie tekstu
Wielkość czcionki

selektor { font-size: rozmiar; }
Rozmiar czcionki możemy podać w trzech jednostkach: pt, px lub em
Waga czcionki

selektor { font-weight: waga; }
Jako "waga" należy wpisać nazwę czcionki, na przykład:
  • normal - czcionka normalna (podstawowa)
  • bold - czcionka pogrubiona
  • 100, 200, 300... - waga czcionki
Przykłady wag:
  • 300
  • 400
  • 500
  • 600
  • 700
