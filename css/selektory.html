Selektory

Czym są selektory ?

W CSS selektor to cześć stylu która poprzedza otwierający nawias klamrowy. Określa on do jakiego elementu odnosi się dana reguła. Selektorami mogą być praktycznie wszystkie tagi HTML takie jak: h1, h2, ol, ul, li, p, span itd.
Najważniejsze selektory:
*

Symbol gwiazdki obierze za cel każdy element znajdujący się na stronie. Wielu deweloperów używa tego triku by wyzerować margin i padding dla całej strony.
* { margin: 0; padding: 0; }
#X

Symbol '#' pozwala nam wybrać cel na podstawie id.Jest to najpopularniejszy sposób wykorzystania, jednak bądź ostrożny używając go.
#x { width: 960px; margin: auto; }
.X

Jest to selektor klasy. Różnica między nim a id jest taka, że klasą możesz wybrać wiele elementów. Użyj klasy jeśli chcesz nadać styl grupie elementów. Ewentualnie, użyj id by znaleźć i wystylizować tylko konkretny element.
.x { color: red; }
X Y

Jednym z najpopularniejszych selektorów jest selektor potomka. Jeśli musisz lepiej sprecyzować cel, używasz właśnie tego. Na przykład, jeśli zamiast zaznaczać wszystkie linki, chcesz wybrać tylko te, które znajdują się w nieuporządkowane liście? Jest to sprecyzowane, gdy używasz selektora potomka.
li a { text-decoration: none; }
X

Jeśli chcesz zaznaczyć wszystkie elementy tego samego typu bez używania id lub klasy użyj selektora typu. Jeśli chcesz zaznaczyć wszystkie nieuporządkowane listy, użyj ul {}.
ul { margin-left: 0; }
X:VISITED / X:LINK

Pseudo-klasy :link używamy by zaznaczyć wszystkie linki które jeszcze nie były kliknięte. Ewentualnie, używamy pseudo-klasy :visited, która, pozwala na dodanie stylu do linka na stronie który był kliknięty, lub odwiedzony.
a:link { color: red; }
a:visited { color: purple; }
X + Y

Ten selektor odwołuje się do sąsiedniego selektora. Wybierze tylko ten element, który występuje bezpośrednio po wcześniej wymienionym elemencie. W tym przypadku, tylko pierwszy paragraf, po każdym ul będzie miał czerwony tekst.
ul + p { color: red; }
X ~ Y

Ogólny selektor następnika jest podobny do X + Y, ale jest mniej ścisły. Kiedy selektor następnika (ul + p) wybierze tylko pierwszy element występujący natychmiast po poprzednim selektorze, ten jest bardziej ogólny.Wybierze, odwołując się do przykładu wyżej, każdy element p jeśli tylko następuje po ul.
ul ~ p { color: red; }
X[TITLE]

Odwołując się do tak zwanego selektora atrybutu,w przykładzie powyżej, wybierze on tylko element z atrybutem title.
a[title] { color: green; }
X:CHECKED

Ta pseudo-klasa obierze za cel tylko zaznaczone elementy - jak przycisk radio lub checkbox.
input[type=radio]:checked
{ border: 1px solid black; }
X:HOVER

Jest to pseudo-klasa akcji użytkownika, która działa na zasadzie najechania kursorem na dany element. W poniższym przykładzie po najechaniu kursorem na element div jego tło zmieni się na czerwony.
div:hover { background-color: red; }
X::PSEUDOELEMENT

Możemy użyć pseudo-elementu (poprzedzonego ::) by wystylizować elementy takie jak pierwsza linia czy pierwsza litera. Zapamiętaj, że muszą one być dodane do elementów blokowych, by zadziałały.
p::first-line { font-size: 20px; }
X::NTH-CHILD(N)

nth-child akceptuje tylko naturalne liczby i zaczyna się od 1. Jeśli chcesz wybrać drugi element z listy użyj li:nth-child(2). Możemy tego użyć by wybrać grupę elementów. Na przykład użyjemy li:nth-child(4) by wybrać co czwarty element listy.
li:nth-child(3) { color: red; }
